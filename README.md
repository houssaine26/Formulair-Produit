# Formulair-Produit
# TP3 - EHEI 2025/2026

## Description

Ce projet est un TP Symfony pour la gestion d'un formulaire de commande de produit.  

### Ce qui a été fait :

- Création d'un **Form Type** `OrderThisType` pour gérer la quantité et la couleur du produit.
- Création d'un **Controller** `OrderThisController` pour afficher la page produit.
- Transformation de la page produit HTML en **Twig template intégrant le formulaire Symfony**
- Intégration de **Bootstrap** pour le style du formulaire et de la page produit.
